# 学习路线

## 前置知识

1. 掌握一门编程语言，对于 LeetCode 而言，推荐学习 Python、Java、C++。
2. 掌握时间复杂度与空间复杂度。
3. 掌握语言自带的数据结构，如数组、对象、栈、堆、队列等。
4. 掌握语言不自带的数据结构，如链表、二叉树、图等。
5. 掌握入门的算法，如递归、排序算法、二分法、宽度优先遍历、深度优先遍历等。
6. 开始刷 LeetCode。
